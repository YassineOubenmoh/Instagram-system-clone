📸 Instagram Clone – Social Media Platform

This project is a social media platform clone inspired by Instagram, developed using Spring Boot and Java 21. It allows users to share posts, interact with content, and build connections in a social networking environment.

Project Structure

Entities / Tables:

Entity	Description
User	Represents the platform users, storing profile information and credentials.
Post	Represents user-created posts including captions, media links, and timestamps.
Comment	Stores comments made by users on posts.
Like	Tracks which users liked which posts.
Save Post	Allows users to save posts for later viewing.
Follower	Maintains the follow/follower relationships between users.
⚡ Features

User Authentication: Sign up, login, and manage user profiles.

Post Management: Create, view, edit, and delete posts.

Engagement: Like, comment, and save posts.

Social Connections: Follow and unfollow other users.

Feed: Display posts from followed users.

🛠️ Technology Stack

Backend: Spring Boot

Language: Java 21

Database: MySQL / H2 (for development/testing)

ORM: Spring Data JPA / Hibernate

Build Tool: Maven
